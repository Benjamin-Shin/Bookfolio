# Bookfolio Mobile

Flutter MVP app for:

- Email auth with Supabase
- Personal shelf list and detail management
- Manual book registration
- Barcode-based ISBN lookup

## Required env

Provide these values before running the app:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
- `BOOKFOLIO_API_BASE_URL`

You can inject them with `--dart-define`.
